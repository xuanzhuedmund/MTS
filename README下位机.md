# Linien 下位机系统

## 系统概述
Linien 下位机系统由FPGA逻辑和Verilog代码组成，实现了锁相放大器的核心信号处理功能，包括数据采集、信号处理、PID控制和自动锁定等。

## 主要组件

### 1. FPGA逻辑（gateware/）
基于Migen框架实现的FPGA逻辑，包括：

#### 1.1 核心模块（linien_module.py）
- **LinienLogic类**：实现核心控制逻辑
  - 初始化CSR（控制状态寄存器）
  - 初始化子模块（调制、扫描、限幅、PID、自动锁定）
  - 连接PID控制逻辑
  - 连接所有组件

- **LinienModule类**：实现完整的信号处理系统
  - 初始化子模块（逻辑、模拟接口、XADC、DeltaSigma调制器等）
  - 连接所有组件
  - 处理信号流

- **RootModule类**：顶层模块
  - 连接处理系统和Linien模块

#### 1.2 逻辑模块（logic/）
- **autolock.py**：自动锁定逻辑
- **autolock_utils.py**：自动锁定工具函数
- **chains.py**：信号处理链
- **cordic.py**：CORDIC算法实现
- **decimation.py**：数据抽取
- **delta_sigma.py**：Delta-Sigma调制
- **filter.py**：滤波器实现
- **iir.py**：IIR滤波器
- **limit.py**：信号限幅
- **modulate.py**：信号调制
- **pid.py**：PID控制器
- **sweep.py**：扫描控制

#### 1.3 底层模块（lowlevel/）
- **analog.py**：模拟接口
- **crg.py**：时钟和复位生成
- **dna.py**：DNA识别
- **gpio.py**：GPIO控制
- **pitaya_ps.py**：Pitaya处理系统接口
- **scopegen.py**：示波器信号生成
- **xadc.py**：XADC接口

### 2. Verilog代码（verilog/）
- **red_pitaya_scope.v**：示波器模块，用于数据采集和触发
- **axi_slave.v**：AXI总线从设备
- **bus_clk_bridge.v**：总线时钟桥接
- **processing_system7_v5_4_processing_system7.v**：处理系统
- **system_processing_system7_0_0.v**：系统顶层
- **system_processing_system7_0_0.xdc**：约束文件

## 信号处理流程

### FPGA逻辑信号流
```
ADC输入 → 快速链处理 → 信号组合 → 限幅 → PID控制 → 限幅 → DAC输出
                                      ↑
                                      |
                               自动锁定逻辑
```

### Verilog示波器信号流
1. **数据采集**：从ADC获取数据
2. **数据处理**：滤波、抽取和平均
3. **触发检测**：检测触发条件
4. **数据存储**：将数据写入BRAM
5. **数据读取**：软件通过总线读取BRAM中的数据

## 核心功能

### 1. 信号处理
- **快速链**：处理高速信号
- **慢速链**：处理经过抽取的信号
- **信号滤波**：去除噪声
- **信号限幅**：防止信号溢出

### 2. PID控制
- **快速PID**：处理实时信号
- **慢速PID**：处理经过抽取的信号

### 3. 自动锁定
- **FPGA端自动锁定**：实现硬件级别的自动锁定
- **稳健锁定算法**：适应不同信号条件

### 4. 调制与扫描
- **信号调制**：生成调制信号
- **扫描控制**：控制信号扫描

### 5. 数据采集
- **示波器功能**：采集和存储数据
- **触发系统**：支持内部、外部和ASG触发
- **数据缓冲**：使用BRAM存储数据

## 技术特点

### FPGA逻辑
- **使用Migen框架**：提供硬件描述能力
- **模块化设计**：便于维护和扩展
- **支持复杂控制逻辑**：实现高级算法

### Verilog代码
- **直接硬件实现**：性能高效
- **专注数据采集**：优化数据采集和存储
- **与系统总线接口**：便于软件控制

## 系统集成
- **上位机与下位机通信**：通过网络协议实现
- **实时数据传输**：确保数据及时更新
- **参数配置**：支持远程配置下位机参数
