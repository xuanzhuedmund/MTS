# Linien 上位机系统

## 系统概述
Linien 上位机系统由多个组件组成，包括图形用户界面、客户端、服务器和通用代码库，共同实现了锁相放大器的控制和监控功能。

## 主要组件

### 1. linien-gui
图形用户界面模块，提供直观的操作界面，包括：
- 主窗口（main_window.py）：系统主界面
- 设备管理（device_manager.py）：设备连接和管理
- 锁定面板（locking_panel.py）：锁相放大器的锁定控制
- 调制扫描面板（modulation_sweep_panel.py）：信号调制和扫描控制
- 优化面板（optimization_panel.py）：系统优化功能
- 光谱面板（spectroscopy_panel.py）：光谱分析功能
- 视图面板（view_panel.py）：数据可视化

### 2. linien-client
客户端模块，负责与服务器通信，包括：
- 连接管理（connection.py）：建立和维护与服务器的连接
- 设备管理（device.py）：设备操作接口
- 远程参数（remote_parameters.py）：远程参数访问

### 3. linien-common
通用代码库，提供共享功能：
- 通信协议（communication.py）：定义客户端和服务器之间的通信协议
- 配置管理（config.py）：系统配置管理
- 通用工具（common.py）：通用工具函数

### 4. linien-server
服务器模块，运行在设备端，包括：
- 自动锁定（autolock/）：自动锁定算法实现
  - 简单锁定（simple.py）：基本锁定算法
  - 稳健锁定（robust.py）：更稳健的锁定算法
- 优化引擎（optimization/）：系统优化功能
- 数据采集（acquisition.py）：数据采集和处理
- 服务器主程序（server.py）：服务器核心逻辑

## 工作流程
1. **用户操作**：用户通过GUI界面发送操作指令
2. **客户端处理**：客户端将指令转换为通信协议格式
3. **服务器执行**：服务器接收指令并执行相应操作
4. **数据反馈**：服务器将执行结果和数据返回给客户端
5. **界面更新**：客户端更新GUI界面，显示结果和数据

## 核心功能
- **锁相放大**：提取微弱信号
- **自动锁定**：自动寻找和锁定信号
- **参数优化**：优化系统参数以获得最佳性能
- **数据可视化**：实时显示信号和频谱
- **远程控制**：支持远程操作和监控

## 技术特点
- **模块化设计**：各组件职责明确，易于维护和扩展
- **跨平台**：支持不同操作系统
- **实时性**：实时数据处理和显示
- **用户友好**：直观的图形界面

## 系统要求
- Python 3.8+
- 相关依赖库（见pyproject.toml）
- 网络连接（用于客户端-服务器通信）
